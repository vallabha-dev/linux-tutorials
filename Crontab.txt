What is crontab?
crontab stands for "cron table", and it's a command-line utility used to schedule repetitive tasks in Unix/Linux systems.

Key Concepts:
Term	Description
cron	A background daemon/service that runs scheduled jobs.
crontab	A file (and command) that defines the schedule of tasks for cron.
cron job	A scheduled command or script to run at specified times.

ðŸ”§ How to Set It Up
Create the directory and script file:

sudo mkdir -p /opt/scripts
sudo vi /opt/scripts/myscript.py

Paste the script below into the file.

#!/usr/bin/env python3

from datetime import datetime

# Log file location
log_file = "/opt/scripts/output.log"

# Get current date and time
now = datetime.now()
timestamp = now.strftime("%Y-%m-%d %H:%M:%S")

# Write timestamp to the log file
with open(log_file, "a") as file:
    file.write(f"Cron job ran at {timestamp}\n")



#Make the script executable:
sudo chmod +x /opt/scripts/myscript.py
python3 /opt/scripts/myscript.py
